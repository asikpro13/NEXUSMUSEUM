# Документация
### Команда запуска проекта
```bash
flutter run --release -t lib/presentation/main.dart
```

### Команда сборки проекта
```bash
flutter build apk --release -t lib/presentation/main.dart
```

**После сборки файл появится по пути:**  
`build/app/outputs/flutter-apk/app-release.apk`

### Окружение
- **Flutter**: 3.41.4
- **Dart**: 3.11.1
- **Версии библиотек**: указаны в `pubspec.yaml`
